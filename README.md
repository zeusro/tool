# 智能文本处理工具 tools.zeusro.tech

一个基于 Tailwind CSS 的炫酷前端工具网页，提供多种文本处理功能。

## 功能特性

### 🔗 URL 解析
- URL 反编码
- 自动拆分并显示所有 Query String 参数
- 显示协议、主机、端口、路径等基本信息

### 📦 JSON 压缩/解压
- 自动检测 JSON 是否已压缩
- 压缩 JSON（移除所有空格和换行）
- 解压缩 JSON（格式化显示）

### 📄 JSON 格式化
- 美化 JSON 格式
- 自动缩进和换行
- 语法高亮显示

### 🌐 UA 分析
- 智能分析 User-Agent 字符串
- 识别浏览器类型和版本
- 识别操作系统和版本
- 识别设备类型（移动/平板/桌面）
- 识别渲染引擎

### 🔐 Base64 工具
- Base64 编码（文本转 Base64）
- Base64 解码（Base64 转文本）
- 自动识别并显示 Base64 图片
- 支持 JPEG、PNG、GIF、WebP、BMP 等格式

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 选择要使用的工具类型
3. 在输入框中输入要处理的内容
4. 点击"处理"按钮查看结果
5. 可以使用"复制结果"按钮复制输出内容

## 技术栈

- **Tailwind CSS** - 现代化 CSS 框架
- **原生 JavaScript** - 无需额外依赖
- **响应式设计** - 支持移动端和桌面端

## 界面特色

- 🎨 渐变背景和玻璃态效果
- ✨ 流畅的动画和过渡效果
- 🌈 多彩的卡片式布局
- 📱 完全响应式设计
